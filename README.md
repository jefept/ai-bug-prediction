# ai-bug-prediction
Machine learning experiments for predicting software defects using code metrics.
This repository contains initial experiments on machine learning
for software defect prediction.

Methodology
Models were evaluated using 5-fold cross validation.
Performance was measured using accuracy, precision,
recall and F1-score.


## Reproducing the Experiment

Clone the repository

git clone https://github.com/jefept/ai-bug-prediction

Install dependencies

pip install -r requirements.txt

Run the training script

python src/train_model.py
